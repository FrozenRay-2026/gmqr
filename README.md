# GMQR

Generate Google Maps URLs and QR codes from clicked map locations in QGIS.

QGIS 上でクリックした地点の Google Maps URL と QRコードを生成し、位置情報共有を簡単に行えるプラグインです。

---

## Features / 機能

- Click map locations / 地図クリック地点取得
- Generate QR codes / QRコード生成
- Generate Google Maps URLs / Google Maps URL生成
- Copy QR images / QR画像コピー
- Save QR images / QR画像保存

---

## Use Cases / 用途例

- Field surveys / 現地調査
- Mobile location sharing / モバイル位置共有
- Outdoor navigation / 屋外ナビゲーション
- GIS data communication / GIS情報共有
