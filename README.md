# GMQR

Generate Google Maps URLs and QR codes from clicked map locations in QGIS.

QGIS 上でクリックした地点の Google Maps URL と QRコードを生成し、位置情報共有を簡単に行えるプラグインです。

---

## Features / 機能

- Copy Google Maps URLs / Google Maps URLコピー
- Open Google Maps in browser / ブラウザで Google Maps を開く
- Copy QR images / QR画像コピー
- Save QR images / QR画像保存

---

## Usage / 使い方

1. Open the GMQR plugin from the toolbar or plugin menu.  
   ツールバーまたはプラグインメニューから GMQR を起動します。

2. Click a location on the QGIS map canvas.  
   QGIS マップキャンバス上の地点をクリックします。

3. A Google Maps URL and QR code will be generated automatically.  
   Google Maps URL と QRコードが自動生成されます。

4. Use the buttons to:
   ボタンを使用して以下の操作ができます。

   - Copy the Google Maps URL  
     Google Maps URL をコピー

   - Open Google Maps in a web browser  
     ブラウザで Google Maps を開く

   - Copy the QR image  
     QR画像をコピー

   - Save the QR image as PNG  
     QR画像を PNG 保存
